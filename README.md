<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ChristianPredoianu/Vanilla-js-quizzer-V2.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   
3. Run dev server
   ```sh
   npm run dev
   ```
4. Build
   ```sh
   npm run build
   ```
